Installazione:
-------------

- Aggiungere al progetto la dipendenza (composer.json):
```
    composer require fi/imapbundle
```
- Aggiungere nel file app/AppKernel.php nella funzione registerBundles;
```
    new Fi\ImapBundle\FiImapBundle(),
```
